# KDF Terminology & Language Style Guide

## Purpose
This document defines mandatory language rules for all KDF materials.

If wording violates this guide, it is non-compliant.

## Core Principles
- Precision over appeal
- No anthropomorphism
- Recording, not reasoning

## Approved Terms
- Knowledge: explicit statements from authoritative sources
- Knowledge Atom: smallest indivisible unit of explicit, evidence-backed content
- Evidence-grounded
- Evidence addressing
- Invalidation
- Conformance

## Restricted Term
Lossless MUST always be used as:
"lossless with respect to structure, semantics, and provenance,
within explicitly defined bounds"

## Prohibited Terms
- understands
- learns
- reasons
- decides
- truth detection
- intelligent extraction
- self-validating
- revolutionary
- cutting-edge

## Tone
Neutral, conservative, formal, restrained.

## Enforcement
Violations must be corrected during review and may block acceptance.
